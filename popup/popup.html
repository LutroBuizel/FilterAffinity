This plugin was originally developed by Forest (Furaffinity / Weasyl) for use on Furaffinity.net.

This plugin as of March 2025 is now maintained by Lutro (Furaffinity). Source code is available here.

It filters out content based on entered artist names and keywords in the submission titles on the Home (artist names only), Browse, and Search pages. If you wish to always display artwork by a certain artist, their names can be allowlisted to do just that.

Hovering over a submission thumbnail will show a Add/Remove option box. Clicking the plus(+) symbol adds the artist's name to the filter list automatically. Clicking the minus(-) symbol removes the artist from the filter list.
